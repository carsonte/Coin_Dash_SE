# Lark卡片（通知卡片）

## 📋 **卡片类型与信息内容**

### **1. 每日绩效报告卡片**

**核心信息**：

- 📅 报告日期
- 📊 总体绩效概览
    - 总信号数、胜率、平均RR
- 🎯 按币种分析
    - BTC/ETH分别的信号数、胜率、盈利点数

### **2. 开仓信号卡片**

**核心信息**：

- 🎯 交易方向与币种（如：BTCUSDT 多头）
- 📈 市场模式与置信度（如：趋势型 85%）
- ⚡ 趋势一致性评分与等级（如：强一致 88分）
- 🎯 交易类型标识（顺势/逆小势/逆大势）
- 💰 关键价格位
    - 入场价、止损价、止盈价
- 📊 风险参数
    - 风险收益比（RR）、置信度
- 🕒 信号有效期
- ⚠️ 特殊风险提示
    - 模式特有风险 + 逆势专项警告
- 💡 建议仓位（轻仓/标准/重仓）
- 🔍 逻辑摘要（AI决策理由）

### **3. 复评调整卡片**

### **3.1 提前平仓通知**

**核心信息**：

- ⚠️ 紧急平仓标识
- 🎯 持仓信息（币种、方向、入场价）
- 💰 平仓价格与盈亏结果
- 📉 平仓原因（AI决策理由）

### **3.2 止损止盈调整通知**

**核心信息**：

- 🔄 调整类型标识
- 🎯 持仓基本信息
- 📊 调整前后对比
    - 原止损/止盈 vs 新止损/止盈
    - 原RR vs 新RR
- 📈 市场变化摘要
- 🧠 调整决策理由
- ⏰ 下次复评时间

### **4. 模式切换预警卡片**

**核心信息**：

- 🔄 模式转换方向
    - 从[当前模式]转向[新模式]
- 📊 转换置信度与预计时间
- 🎯 受影响币种
- ⚠️ 风险等级变化
- 💡 操作建议
    - "谨慎开新仓"、"注意止损调整"等
- 📈 关键监控指标

### **5. 异常告警卡片**

**核心信息**：

- 🚨 异常类型标识
    - API连接异常/数据校验异常/风控熔断等
- ⏰ 发生时间与持续时间
- 📍 影响范围
    - 哪些功能受限（新信号/复评/交易执行）
- 🔧 是否恢复
